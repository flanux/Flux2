# Branch Dashboard

Branch management interface for banking system operations.

## Tech Stack
- React 18 + TypeScript
- Vite
- Material-UI (MUI)
- Zustand (State Management)
- React Hook Form + Yup
- Axios
- Recharts

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Docker
```bash
docker build -t branch-dashboard .
docker run -p 3001:80 branch-dashboard
```
