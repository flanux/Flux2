# Central Bank Portal

Central bank oversight and management interface for the banking system.

## Tech Stack
- React 18 + TypeScript
- Vite
- Material-UI (MUI)
- MUI X Data Grid & Charts
- Zustand (State Management)
- React Hook Form + Yup
- Axios
- Recharts

## Features
- System-wide monitoring
- Branch performance analytics
- Regulatory compliance
- Policy management
- Interest rate management
- Liquidity monitoring
- Audit trails

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Docker
```bash
docker build -t central-bank-portal .
docker run -p 3002:80 central-bank-portal
```
